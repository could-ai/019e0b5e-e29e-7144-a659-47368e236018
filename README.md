# Learn Korean App

A cross-platform Flutter application for learning Korean, featuring interactive flashcards for Hangul, basic vocabulary, and common phrases.

## Features
- **Learning Categories**: Browse topics like Hangul basics, greetings, and numbers.
- **Interactive Flashcards**: Flip cards to see translations, read pronunciations, and learn through spaced repetition style.
- **Responsive Design**: Works seamlessly on mobile, tablet, desktop, and web.

## Tech Stack
- Flutter
- Dart
- Material 3 Design

## Setup
1. Ensure Flutter is installed.
2. Run `flutter pub get` to fetch dependencies.
3. Run `flutter run` to launch the app on your preferred device.
